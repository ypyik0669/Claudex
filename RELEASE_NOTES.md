# Claudex v0.1.0

Initial public preview of Claudex, a desktop coding-agent shell for Claude Code CLI with a Codex-inspired workspace.

## Downloads

- Windows: use the `.exe` installer or `.zip` portable build.
- macOS: use the `.dmg` installer or `.zip` build.

## Notes

- Claude Code mode requires Claude Code CLI installed and authenticated on the user's machine.
- Direct API mode can be configured inside Settings.
- API keys are stored locally by the app and are not bundled in release assets.
- macOS builds are unsigned in this preview, so Gatekeeper may require manual approval on first launch.
