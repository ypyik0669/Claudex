# Codex App 行为对齐执行计划

## 当前方向

Claudex 采用中文桌面工作区：左侧导航、中间聊天、右侧工具、底部上下文面板。所有可点击入口都应连接真实本地状态或真实 Claude Code CLI 命令。

## 执行重点

- 降低默认界面噪音。
- 保留 Claude Code 原生能力入口。
- 设置页按真实运行时能力展示，不做假控件。
- 插件和 MCP 以 CLI 输出为准。
- 文件编辑必须有保存前差异审查。
- 文档、错误提示和发布说明中文化。

## 验证

- 本地构建。
- Electron QA 脚本。
- GitHub Actions Release 构建。
